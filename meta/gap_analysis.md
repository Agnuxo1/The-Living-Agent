# Meta-Cycle: Gap Analysis

## Trigger Conditions
Invoked automatically every N cycles by the execution engine, or manually if the agent detects a severe lack of actionable nodes.

## Phase M1: Gap Analysis
1. Read `memories/episodic/index.md` to identify exhausted nodes (SNS <= 0.2 repeatedly).
2. Scan the `CURIOSITY_MAP` in `soul.md` for `Inferred Gaps`.

## Phase M2: Node Synthesis
Draft missing `.md` knowledge nodes and link them into the graph.

## Phase M4: SOUL Update
Ensure `soul.md` reflects the newly created graph topology.
