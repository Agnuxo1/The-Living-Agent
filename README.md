# The Living Agent 🧬
### Autonomous Research & Discovery Engine (P2PCLAW Silicon Layer)

Welcome to **The Living Agent**, a fully autonomous research AI designed to operate 24/7, exploring the frontiers of bridge engineering, biological computing, and physics.

This agent uses a **Stochastic Semantic Finite State Machine (S²FSM)** to navigate a "Text-as-Code" knowledge graph, evolving its own skills and understanding over time.

---

## 🚀 Quick Start (One-Click)

1. **Download the Repository** (or `git clone https://github.com/Agnuxo1/The-Living-Agent`)
2. **Download the Brain**:
   - Model: `Qwen3.5-9B-UD-Q3_K_XL.gguf` (5.05 GB)
   - [Download Link](https://huggingface.co/unsloth/Qwen3.5-9B-GGUF/resolve/main/Qwen3.5-9B-UD-Q3_K_XL.gguf)
3. **Download the Body**:
   - [KoboldCPP](https://github.com/LostRuins/koboldcpp/releases/latest) (Download `koboldcpp.exe`)
4. **Run**:
   - Open `koboldcpp.exe` and load the Qwen model on port **5001**.
   - Double-click **`start.bat`**.

---

## 🛠 Features
- **24/7 Autonomy**: Researches, synthesizes papers, and expands its knowledge graph without human intervention.
- **Text-as-Code**: The agent's identity and logic are stored in human-readable Markdown files (`soul.md`, `skills/`).
- **Semantic Novelty (SNS)**: Automatically calculates the novelty of its discoveries to prioritize high-impact research.
- **P2PCLAW Integration**: Ready to sync with the Silicon infrastructure.

## 📁 Repository Structure
- `agent_v2_production.py`: The central execution engine.
- `soul.md`: The agent's persistent identity and memory.
- `knowledge/`: The evolving graph of human and AI discoveries.
- `memories/`: Historical research papers and episodic traces.
